place the coco folder into data/ folder
download the raw images from here: http://mscoco.org/
and place them all into coco/train2014 and coco/val2014 . 
You only have to do this if you wish to visualize the predictions:

here files descriptions below: 
coco_dict_path_split: contains the dictionary used coco captions, the path where each image sits, and the split each image is assigned 

coco_labels.h5: contains labels array, label_start_idx, label_end_idx, label_length
